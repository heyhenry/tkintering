# Objective: Simple Calculator

Key Features:

- Display Input: A label or entry field to show the current input.
- Buttons for Digits: Buttons for numbers 0-9 and arithmetic operations (addition, subtraction, multiplication, division).
- Calculate Result: A button to calculate the result of the expression entered.
- Clear Input: A button to clear the current input and reset the calculator.
- Display Result: Show the result of the calculation in the same display area.
